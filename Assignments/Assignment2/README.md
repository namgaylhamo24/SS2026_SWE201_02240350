# Student Productivity App — Assignment 2

This is a frontend-only React Native (Expo) app built for SWE201 Assignment 2. It demonstrates multi-screen navigation, animations, and gesture interactions.

Run locally:

```bash
cd Assignments/Assignment2
npm install
npm run start
```

Key features:
- Stack + Tab navigation
- Animated Detail screen (slide + fade)
- Gesture-driven draggable element (Animation demo)
- Reusable `Card` component
